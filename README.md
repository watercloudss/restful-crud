# restful-crud
springboot国际化处理简单实例<br>
点击页面上的按钮“中文”或“English”进行语言的切换<br>
只在登录页实现了国际化，还有一些增删改查的功能，没有连接数据库，使用了restful风格，当做springboot的一个练习<br>
